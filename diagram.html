The Critical Rule

A number alone (e.g. 5) shows full task detail ‚Äî it never marks it complete. A number + "done" (e.g. 5 done) marks it complete. Two completely separate actions. This rule prevents accidental completions.

Never skip this

1. Find most recent YYYY-MM-DD.md ‚Äî 2. Read it in full ‚Äî 3. Confirm to user with brief summary of open tasks ‚Äî 4. Display the shortcuts list. Every session, automatically.
